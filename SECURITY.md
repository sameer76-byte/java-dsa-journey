# Security Policy

## Supported Versions

This is a personal learning repository for Java and Data Structures. Security updates are generally applied only to the latest version of the code in the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| < Main  | :x:                |

## Reporting a Vulnerability

I take the security of my code seriously, even in a learning environment. If you find any security vulnerabilities (such as exposed credentials or insecure logic), please do not report them via public issues.

Instead, please report them by:
1. **GitHub Private Vulnerability Reporting:** Use the "Security" tab on this repository to report a vulnerability privately.
2. **Email:** You can reach out to me via the email listed on my GitHub profile.

I will do my best to respond within 48–72 hours to acknowledge the report and discuss a fix.

## Disclaimer
This repository is for educational purposes. While I strive for best practices, the implementations here are part of a learning journey and should be used with caution in production environments.
