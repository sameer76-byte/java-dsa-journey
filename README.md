## 🛠️ Setup & Usage

**Prerequisites:** Java 17+ and any IDE (IntelliJ IDEA recommended)
```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/java-dsa-journey.git
cd java-dsa-journey

# Compile and run any file
javac 01-basics/01-variables-datatypes/Main.java
java -cp 01-basics/01-variables-datatypes Main
```
